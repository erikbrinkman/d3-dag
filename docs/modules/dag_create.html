Module dag/create

Three built-in methods exist to turn existing raw data into Dags:

  1. HierarchyOperator - when the data already has a dag structure.
  2. StratifyOperator - when the dag has a tabular structure, referencing parents by id.
  3. ConnectOperator - when the dag has a link structure and is specified as pairs of nodes.
