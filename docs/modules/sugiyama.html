cachedNodeSize

  • cachedNodeSize<N, L>(nodeSize: SugiNodeSizeAccessor<N, L>, check?: boolean): readonly [CoordNodeSizeAccessor<N, L>, CoordNodeSizeAccessor<N, L>]
    • Defined in src/sugiyama/index.ts:301

    Verify, cache, and split the results of an SugiNodeSizeAccessor into an x and y CoordNodeSizeAccessor.

    This allows you to split an SugiNodeSizeAccessor into independent x and y accessors, while also caching the result to prevent potentially expensive computation from being duplicated.

    The only real reason to use this would be to run the steps of sugiyama independently.

sugiyama

  • sugiyama(...args: never[]): DefaultSugiyamaOperator
    • Defined in src/sugiyama/index.ts:554

    Construct a new SugiyamaOperator with the default settings.

    example
    const dag = hierarchy()(...);
    const layout = sugiyama().nodeSize(d => d === undefined ? [0, 0] : [d.width, d.height]);
    layout(dag);
    for (const node of dag) {
      console.log(node.x, node.y);
    }
