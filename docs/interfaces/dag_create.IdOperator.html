  • IdOperator(d: Datum, i: number): string
    • Defined in src/dag/create.ts:419

    The interface for getting a node id from data. The function must return an appropriate unique string id for given datum. This operator will only be called once for each input.

    i will increment in the order data are processed.

    This is used in StratifyOperator.id, ConnectOperator.sourceId, and ConnectOperator.targetId.

    Parameters

    • d: Datum
    • i: number
