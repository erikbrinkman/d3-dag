    A LayeringOperator that restricts the layout to have a maximum width.

    Assigns every node a layer such that the width, not counting dummy nodes, is always less than some constant. This can result in tall graphs, but is also reasonably fast. If the max width is set to zero (the default), the width will instead be set to the square root of the number of nodes.

    This method is reasonably fast, but can results in long edges which make the decrossing phase more difficult. Also note that setting width to 1 is roughly equivalent to topological layering.

    Create with coffmanGraham.
