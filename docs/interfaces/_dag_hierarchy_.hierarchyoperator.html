Callable

  • __call(...data: NodeDatum[]): Dag<DagNode<NodeDatum, LinkDatum>>
    • Defined in src/dag/hierarchy.ts:84

    Construct a DAG from the specified root nodes. Each root node must be an object representing a root node. For example:

    {
  "id": "Eve",
    "children": [
    {
      "id": "Cain"
    },
    {
      "id": "Seth",
      "children": [
      {
        "id": "Enos"
      },
      {
        "id": "Noam"
      }
      ]
    },
    {
      "id": "Abel"
    },
    {
      "id": "Awan",
      "children": [
      {
        "id": "Enoch"
      }
      ]
    },
    {
      "id": "Azura"
    }
  ]
}

    Node ids must be unique, and can't contain the null character '\0'.
